# 更新日志

按主题记录本轮对系统的优化与新功能。运行方式不变（双击桌面图标 / `启动.bat` / `npm start`），源码改为 `src/` 模块化 + `node build.mjs` 构建（详见 `src/README.md`）。

## 2026-07 本轮优化

### 🚀 性能与工程
- **按视图渲染**：`render()` 只重建当前可见视图，不再每次操作都重建全部 12 个视图（大列表下明显更跟手）。
- **持久化防抖**：多次连续改动合并成一次写盘，并在关闭/隐藏页面时兜底落盘，既省开销又不丢数据。
- **筛选框输入防抖**：潜客/搜索/会话筛选连续输入合并渲染，不再每敲一个字符就重建整张列表。
- **模块化构建**：`app.js` 拆成 `src/` 下 8 个模块，`build.mjs` 拼接生成（逐字节等价），自动打版本戳防浏览器缓存旧版。

### 🤖 AI 引擎
- **多服务商支持**：新增「云端大模型」——Claude / OpenAI ChatGPT / DeepSeek / 通义千问 / Kimi / 智谱 GLM，或「自定义」填任意 OpenAI 兼容接口。默认省钱用 DeepSeek，Claude 仅联网功能需要。
- **上下文喂足**：写信/回复提示词自动带上具体产品定位、术语、HS 编码、买家画像、客户类型与**产品知识库**，输出更贴产品、不编造（此前回复建议根本没带知识库）。

### 🧭 更好上手 / 日常提速
- **首次欢迎向导**：新用户第一次打开给两条明路——「一键体验」用演示数据跑到发信队列看效果，或「填我的产品直接开始」。
- **今日待办工作台**：控制台顶部聚合每天要处理的事（待审批/到期待发/新回复/到期跟进/Agent 待审批/超期未备份），一键直达或直接处理；Webhook 模式下还可一键「拉取新回复 / 拉取送达状态」。
- **优先联系名单**：数据分析看板按机会分排出「今天先追谁」Top5（已回复 > 已打开 > 高分），点一行直达该客户。

### 📨 发信域名保护（到达率）
- **退订说明**：所有开发信（模板 + AI 写信）自动附一句轻量退订说明，回复 unsubscribe 即被识别退订并拉黑。
- **送达/退信回传**：新增「发送状态回传 Webhook」，同步 送达/打开/退信/投诉；**硬退信与投诉自动进黑名单**，不再往坏地址发。
- **退信率警告**：数据分析看板显示退信率，样本≥10 且≥5% 时红字提示先停量。
- **垃圾词预检**：发信预检徽章扫描易进垃圾箱的词/排版（guarantee、act now、click here、$$$、连续感叹号、多个全大写词……，排除 FOB/MOQ 等外贸缩写），命中提示改写。

### 💼 报价转化包（询盘 → 报价 → 成交）
- **产品库**：管理页维护常卖型号（型号/英文品名/MOQ/参考价/单位/箱规/认证），一处维护三处受益——AI 回复引用真实参数、报价单直接选行、开发信卖点更准。
- **报价单生成器**：CRM 页「生成报价单」——选客户 + 勾产品行（或加自定义行）+ 贸易条款/币种/有效期 → 生成专业英文报价单（自动编号 Q-日期-序号），可**打印/存 PDF** 或**一键复制文本版**粘进回信。
- **CRM 联动**：生成即自动推进到「报价」阶段、管道金额换成真实报价额；3 天后引用报价编号的跟进邮件自动排队（待审批）——报价不再石沉大海。

### ⏰ 发送时机（按市场时区）
- 发信队列每封待发邮件显示**发送时机徽章**：客户当地此刻几点、最佳发送对应的北京时间段（当地上午 9-11 点）；当地正值上午显示"现在发正合适"，深夜则提醒换时段。
- Webhook 发信 payload 附带 `recipient_utc_offset / recipient_local_time / suggested_window_beijing` 字段，n8n 可按此定时投递，开信率更高。覆盖 40+ 常见市场（不处理夏令时，按代表时区近似）。

### 💾 数据安全
- **老邮件瘦身**：设置「数据与备份」可一键把 30 天前发出、客户未回复的已发邮件正文归档清空（分析统计不受影响），缓解 localStorage 5MB 上限。
- **备份进待办**：超 7 天未备份直接在今日待办出一行，一键导出 JSON。
